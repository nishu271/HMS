Hostel Management System

The Hostel Management System is a an online portal to make the day to day Hostel Activities more efficient. It is going to have 4 submodules :

Security

The existing security system in place is pretty traditional. Students have to fill an entry at the security while entering or leaving the hostel. Guests have to do the same as well. There is no online record for this and if some security log gets misplaced, you lose the complete logging information for a large period. With a security portal the students/security guards can enter the student logs directly into the database. This system can later be made more efficient by using students' id cards instead of manually entering the details.
